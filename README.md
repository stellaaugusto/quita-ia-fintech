# Quita.IA

Fintech desenvolvida para auxiliar usuários no planejamento estratégico e inteligente da quitação de dívidas.

## Funcionalidades

- Cadastro de dívidas
- Visualização consolidada de débitos
- Priorização automática por taxa de juros (Método Avalanche)
- Simulação de cenários de pagamento
- Índice de Sufoco Financeiro (ISF)
- Recomendações inteligentes para redução do endividamento

## Tecnologias Utilizadas

- HTML5
- CSS3
- Tailwind CSS
- Git
- GitHub

## Sobre o Projeto

Quita.IA foi desenvolvido como parte do curso de Análise e Desenvolvimento de Sistemas da FIAP.

Esta entrega corresponde à implementação da interface web responsiva do sistema, seguindo os requisitos definidos nas etapas anteriores do projeto.
